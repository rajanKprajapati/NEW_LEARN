7ï¸âƒ£ super() Function
âœ… Meaning
super() is used to call parent class methods or constructors inside child class.

Why super() is useful?
âœ… Avoid writing parent class name directly
âœ… Helps in multiple inheritance (MRO)
âœ… Calls parent constructor properly

âœ… super() with method
class Parent:
    def show(self):
        print("Parent show")

class Child(Parent):
    def show(self):
        super().show()
        print("Child show")

c = Child()
c.show()
Output
Parent show
Child show
âœ… super() with constructor
class Parent:
    def __init__(self, name):
        self.name = name

class Child(Parent):
    def __init__(self, name, age):
        super().__init__(name)
        self.age = age

c = Child("Rajan", 19)
print(c.name, c.age)
